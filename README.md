# QA Practice Repository 2

This is a second mock take-home QA assignment for practice.

## Goal
Practice working with a GitHub repository, reading instructions carefully, writing answers in markdown files, and optionally completing a small Python task.

## Structure
- `INSTRUCTIONS.md` — rules and submission expectations
- `tasks/` — assignment prompts
- `resources/` — API description and sample payloads
- `answers/` — write your answers here
- `reference_solution/` — example answers for self-checking after you finish

## Practice rules
For realistic practice:
- Read the instructions first.
- Do not open `reference_solution/` until you finish.
- Write answers in the matching files under `answers/`.
- Commit changes after each task or when you finish a full task.
